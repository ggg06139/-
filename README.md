# "으뜸 파이썬" 교재 예제 풀이

### 사용 언어
파이썬


### 단원 별 교재 내용

2단원: 변수와 연산자

3단원: 제어문

4단원: 함수와 입출력

5단원: 리스트

6단원: 딕셔너리, 튜플, 집합

7단원: 모듈과 활용

8단원: 예외 처리와 파일

9단원: 클래스와 객체 지향 프로그래밍

10단원: 람다 함수와 리스트 축약
